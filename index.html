Section 1: Introduction to HTML

a. What is HTML, and why is it important in web development?

  • HTML serves as the backbone of every website, providing the essential structure upon which all web pages are built.

    • b. What are HTML extensions, and how do they function?

  • HTML files are typically saved with extensions like .html or .htm, which tell web browsers to interpret and display them as web pages.
  • HTML extensions can also refer to additional tools or technologies that expand HTML's capabilities, such as developer browser add-ons or HTML5 features that enhance website functionality.

    • Section 2: HTML Elements and Tags

    a. What is the difference between an HTML element and an HTML tag?

  • HTML is the language used to create and structure web pages, while an HTML tag is a specific element within that language used to define parts of the content (like < p > for paragraph or < h1 > for heading).
  • HTML refers to the entire framework or code used for building websites, whereas an HTML tag is one of the building blocks within that code that tells the browser how to display certain elements.

    • b. List and describe the purpose of the following tags:

  • < html > - Defines the beginning and end of an HTML document. It wraps all the content on the page.
  • < head > - Contains meta-information about the document (e.g., title, character set, links to CSS or JavaScript).
  • < title > - Sets the title of the web page, which appears in the browser tab.
